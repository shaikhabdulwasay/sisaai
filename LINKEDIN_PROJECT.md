**SISA AI** is a fully offline, privacy-first conversational AI chat application that I designed and built from scratch. The app features a high-end futuristic glassmorphic UI and runs a quantized large language model (LLM) entirely on the local machine — no cloud API, no data leakage, no subscription fees.

**What the project does:**
- Provides a sleek, browser-based chat interface powered by a local AI model
- Uses Ollama (an open-source LLM runtime) to serve the Qwen 3.5 (0.8B) language model locally
- Maintains full conversation history within each session, visible in a real-time sidebar
- Supports a one-click conversation reset to start fresh at any time
- Runs 100% offline — your conversations never leave your device

**Key Technical Achievements:**
- Engineered a custom multi-layer CSS theming system featuring glassmorphism (backdrop-blur, RGBA transparency layers), the futuristic Orbitron display font, and smooth hover/transition animations — all injected dynamically via Streamlit's `st.markdown()` API
- Integrated Ollama's Python SDK to perform synchronous inference calls to a locally hosted quantized LLM, with robust error handling to surface clear diagnostics when the model server is offline
- Implemented reactive session-state management using Streamlit's `st.session_state` to persist and render the full conversation history without page reloads
- Designed a responsive dual-pane layout: a sidebar showing a live scrollable log of recent exchanges and a main area for real-time chat
- Configured a custom Streamlit theme (`config.toml`) with a near-black background (#0a0a0a), pure white primary accents, and a dark secondary surface — delivering a consistent futuristic aesthetic across all components

**Why this project matters:**
As AI assistants become mainstream, privacy concerns around cloud-based tools are growing. SISA AI demonstrates that anyone can run a capable, interactive AI assistant entirely on their own hardware. This project showcases my ability to combine backend AI integration, frontend UI design, and full-stack Python application development into a polished, production-ready product.
